# 🎓 SGT University AI Chatbot

An AI-powered chatbot for answering student queries about admissions, courses, fees, campus, and placements.

## 🚀 Features

- Floating AI Buddy chatbot
- Modern white & pink UI
- Background image slider
- AI responses using Ollama (offline LLM)
- Flask backend

## 🛠 Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- Ollama (phi3 / Gemma)
- Machine Learning / RAG
